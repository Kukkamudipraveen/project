Let us Learn about the Inheritance in Java using Html:

  • Inheritance:

    Inheritance can be defined as a feature of obtaining the properties and behaviour of an existing class by a derived class.i.e; A child class(derived class) can acquire the properties of parent class(existing class).It can be done by using a keyword "extends".

    There are 5 types of inheritances

    1. Single Inheritance
    2. Multi-Level Inheritance
    3. Hierarchical Inheritance
    4. Multiple Inheritance
    5. Hybrid Inheritance

  1. Single Inheritance:

    It is a type of inheritance in which a single derived class(sub class) and a single existing class(super class) are present.

    Program Outline :

    class Old
    {
    public void method1()
    {
    System.out.println("This is a single inheritance.");
    System.out.println("Message from class Old.");
    }
    }
    class Inheritance extends Old
    {
    public static void main(String[] args)
    {
    Inheritance I=new Inheritance();
    I.method1();
    }
    }
    Program Output :

    This is a single inheritance.
    Message from class Old.

  2. Multi-level inheritance:

    It is a type of inheritance in which a super class,an intermediate class and a derived class are present.i.e; Three classes,The derived class(child class) obtains the properties and behaviour from the intermediate class(parent class), which obtain the properties and behaviour of the super class.

    Program Outline :

    class Old
    {
    public void method1()
    {
    System.out.println("This is a multi-level inheritance.");
    System.out.println("Message from class Old.");
    }
    }
    class Intermediate extends Old
    {
    public void method2()
    {
    System.out.println("Message from class Intermediate.");
    }
    }
    class Inheritance extends Intermediate
    {
    public static void main(String[] args)
    {
    Inheritance I=new Inheritance();
    I.method1();
    I.method2();
    }
    }
    Program Output :

    This is a multi-level inheritance.
    Message from class Old.
    Message from class Intermediate.

  3. Hierarchical Inheritance:

    It is a type of inheritance in which two sub classes are present and the two sub classes obtain the properties and behaviour of same super class.i.e; one super class and two sub classes.

    Program Outline :

    class Old
    {
    public void method1()
    {
    System.out.println("This is a Hierarchical inheritance.");
    System.out.println("Message from class Old.");
    }
    }
    class Inherit extends Old
    {
    public void method2()
    {
    System.out.println("Message from class Inherit.");
    }
    }
    class Inheritance extends Old
    {
    public static void main(String[] args)
    {
    Inheritance I=new Inheritance();
    I.method1();
    System.out.println("");
    Inherit In=new Inherit();
    In.method1();
    In.method2();
    }
    }
    Program Output :

    This is a Hierarchical inheritance.
    Message from class Old.
    This is a Hierarchical inheritance.
    Message from class Old.
    Message from class Inherit.

  4. Multiple Inheritance:

    It is a type of inheritance in which sub class acqire the properties and behaviour of super classes.i.e; Two super classes and one sub class

    Multiple inheritance cannot not be performed by using classes.

  5. Hybrid Inheritance:

    It is a type of inheritance in which two classes acquires the properties and behaviour of their super class and later they try to share their properties and behaviour to their sub class.It is a combination of multiple and hierarchical inheritances.

    Hybrid inheritance cannot not be performed by using classes.
